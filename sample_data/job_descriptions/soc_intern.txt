Job Title: SOC Intern
Department: Cyber Security / Security Operations Centre (SOC)
Reports To: SOC Team Lead / Security Manager
Experience: Freshers

Job Summary:
The Security Analyst Intern will assist the Security Operations Centre (SOC) team in monitoring, analysing, and responding to cybersecurity alerts and incidents. This internship provides hands-on exposure to real-world cybersecurity operations, security monitoring tools, threat detection techniques, and incident response processes.
The intern will work closely with SOC analysts to learn security operations practices, analyse logs, investigate alerts, and support the organization's cybersecurity defence efforts.

Key Responsibilities:
1. Security Monitoring Support: Assist in monitoring security alerts generated by SIEM, EDR, firewalls, and other security tools. Help review and categorize security alerts and events.
2. Log Analysis: Assist in analysing system and network logs for suspicious activities. Review logs from Windows/Linux systems, network devices, and security tools.
3. Alert Investigation Assistance: Support analysts in investigating low-level security alerts. Help validate true positives vs false positives.
4. Threat Intelligence Research: Conduct research on emerging threats and vulnerabilities. Identify indicators of compromise (IOCs) such as malicious IPs, domains, and hashes.
5. Vulnerability Monitoring: Assist with vulnerability scan reviews and track remediation efforts.
6. Documentation: Document security incidents and daily SOC activities.

Required Qualifications:
- Education: Currently pursuing or recently completed a bachelor’s degree in Cybersecurity, Computer Science, Information Technology, or related.
- Cybersecurity Concepts: Basic understanding of cybersecurity principles, threats, malware basics, phishing attacks.
- Networking: TCP/IP, DNS, HTTP/HTTPS, Firewalls, VPN.
- Operating Systems: Windows, Linux.
- Security Tools: Exposure to SIEM platforms, Antivirus/EDR tools.
- Certifications (Preferred/Optional): CompTIA Security+, Google Cybersecurity Certificate, Certified SOC Analyst (CSA), CEH (Certified Ethical Hacker).
